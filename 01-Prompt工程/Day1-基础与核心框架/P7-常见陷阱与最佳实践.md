# P7：常见陷阱与最佳实践 + 小项目

## 目标
系统梳理 Prompt 工程的常见问题，完成一个 Prompt Playground 小项目

## 第一部分：常见陷阱

### 陷阱 1：指令冲突

```
错误： System: "你是一个创意写手，回答要简洁"
   → "创意"和"简洁"是矛盾指令，模型无法同时满足

正确： System: "你是一个创意写手，用诗意的语言但控制在3句话以内"
   → 明确的、可量化的约束
```

**原则**：指令要一致，避免内耗；用具体数字替代模糊描述。

### 陷阱 2：上下文过长/过短

| 问题 | 表现 | 解决 |
|------|------|------|
| 过长 | 中间信息被忽略（Lost in the Middle） | 重要信息放开头和结尾 |
| 过短 | 缺乏上下文，模型自由发挥 | 提供足够背景和示例 |

**Lost in the Middle 现象**：在长上下文中，模型倾向于记住开头和结尾的信息，中间部分容易被忽略。

### 陷阱 3：过度指令

```
错误： "你必须、一定要、绝对不要、千万不能、无论如何都要..."
   → 过多的强调词反而降低注意力

正确： 正面指令 + 明确负面指令，各一条
```

### 陷阱 4：隐含假设

```
错误： "帮我修复这个 bug" → 模型可能不知道代码背景
正确： "以下 Python 代码在 Python 3.12 运行，使用 FastAPI 框架，问题是在调用 /users API 时返回 500 错误。代码：..."
```

### 陷阱 5：模型幻觉

| 原因 | 表现 | 应对 |
|------|------|------|
| 知识截止日期后的事件 | 编造答案 | 提供参考信息或使用 RAG |
| 训练数据中少见的知识 | 自信地胡说 | 要求模型标注不确定程度 |
| 暗示性问题 | 迎合用户 | 避免暗示性提问 / 使用中性措辞 |

### 逆向提问法

当不确定该提供什么信息时，可让 AI 先提问：

```
我是一名程序员，想开发一款帮助用户管理个人待办事项的 App，
但我不知道如何制定商业计划。

请你扮演一名经验丰富的创业导师。为了帮我制定一份详尽的商业计划书，
请向我提出一系列关键问题。一次只问一个问题，直到认为收集了足够的信息为止。

如果明白，请提出第一个问题。
```

> 场景：用户对任务领域不熟悉、不知道需要提供什么背景信息、需求模糊时。AI 通过问答逐步缩小范围，最终给出精准答案。

### 摘要 vs 提取

| 维度 | 摘要（Summarization） | 提取（Extraction） |
|------|----------------------|-------------------|
| 输出内容 | 重新组织语言，保留核心信息 | 从原文中直接复制片段 |
| 语言风格 | 可能与原文不同 | 与原文一致 |
| 适用场景 | 长文档浓缩、多文档合并要点 | 关键信息抓取、证据收集 |
| 控制方式 | 控制长度、聚焦主题 | 指定提取字段、格式 |
| 模型难度 | 较高（需要理解 + 重组） | 较低（需要定位 + 复制） |

> 原则：需要保持原意的用提取（法律、医疗报告），需要理解后再表达的用摘要（周报、会议纪要）。

### 陷阱类型总览

| 类别 | 典型问题 | 严重程度 |
|------|---------|---------|
| 指令冲突 | System 中同时要求"创意"和"简洁" | 高 |
| 知识盲区 | 模型不知道但自信回答了（幻觉） | 高 |
| 上下文过载 | Lost in the Middle，中间信息被忽略 | 中 |
| 过度指令 | 一堆"必须"、"绝对要"反而降低注意力 | 中 |
| 隐含假设 | 默认模型知道代码上下文 | 高 |
| 缺乏兜底 | 模型无法回答时编造答案而非承认不知道 | 高 |

## 第二部分：Prompt 评估框架

在优化 Prompt 之前，先定义"好 Prompt"的标准。推荐 5 维度评估表：

| 维度 | 权重 | 评估方式 | 得分标准 |
|------|------|---------|---------|
| 准确率 | 30% | 测试集预期输出匹配率 | 正确答案数 / 总测试数 |
| 完整性 | 20% | 是否遗漏需求中的要点 | 覆盖全部要求的比例 |
| 一致性 | 20% | 同一 Prompt 多次运行结果是否稳定 | 关键结果一致的比例 |
| 安全性 | 15% | 对注入攻击的抵抗能力 | 未被绕过的测试数 / 总测试数 |
| 体验 | 15% | 输出格式、语气、可读性 | 用户主观评分（1-5） |

实践中，使用第三部分的 `Evaluator.run_test` 进行评估：

```python
from evaluator import Evaluator

evaluator = Evaluator()
result = evaluator.run_test(
    test_cases=test_cases,
    system_prompt="你的 system prompt",
    judge_function=lambda out, exp: exp in out,
    verbose=True
)
```

> 原则：没有评估就没有优化。每次改 Prompt 前先跑一次基线，改完再跑一次，确保变化是有价值的。

### 调试 Prompt 的标准流程

```
1. 明确任务目标
2. 写第一个版本
3. 测试 5~10 个不同用例
4. 分析错误模式（格式错误？内容错误？遗漏信息？）
5. 针对性修改 Prompt
6. 重复 3~5 直到满意
7. 固定为模板
```

### 统一最佳实践清单

- **System Prompt 越具体越好**：给出角色、目标、规则、输出格式、约束条件
- **先写后删**：先写详细的 Prompt，再逐步精简到核心
- **A/B 测试**：任何改动都要对比测试
- **示例 > 指令**：想让模型做什么，不如直接给它看一个例子

#### llm_client.py

```python
import os, json, time, logging
from typing import Any, Callable
from openai import OpenAI
from dotenv import load_dotenv

load_dotenv()

class LLMClient:
    """LLM API 客户端封装，统一管理对话/流式/工具调用/重试/用量统计"""
    def __init__(
        self,
        api_key: str|None = None,
        base_url: str = "https://api.deepseek.com",
        model: str = "deepseek-v4-flash",
        max_retries: int = 3,
    ):
        # API 密钥优先用传入值，fallback 到环境变量
        self.client = OpenAI(
            api_key=api_key or os.getenv("DEEPSEEK_API_KEY"),
            base_url=base_url
        )
        self.model = model
        self.max_retries = max_retries
        # 累计用量，便于后续成本监控
        self.total_prompt_tokens = 0
        self.total_completion_tokens = 0
        self.call_count = 0
        logging.basicConfig(level=logging.INFO, format='%(asctime)s - %(levelname)s - %(message)s')
        self.logger = logging.getLogger(__name__)

    def chat(
        self,
        messages: list[dict],
        temperature: float = 0.7,
        top_p: float = 0.9,
        max_tokens: int|None = None,
        tools: list|None = None,
        tool_choice: str|None = "auto",
        response_format: dict|None = None,
        stream: bool = False,
        reasoning_effort: str|None = None,
    ) -> Any:
        # 核心方法：透传所有 OpenAI 参数，只传非 None 的 kwargs
        kwargs = {
            "model": self.model,
            "messages": messages,
            "temperature": temperature,
            "top_p": top_p,
        }
        if max_tokens:
            kwargs["max_tokens"] = max_tokens
        if tools:
            kwargs["tools"] = tools
            kwargs["tool_choice"] = tool_choice
        if response_format:
            kwargs["response_format"] = response_format
        if stream:
            kwargs["stream"] = True
        if reasoning_effort:
            kwargs["reasoning_effort"] = reasoning_effort
            kwargs["extra_body"] = {"thinking": {"type": "enabled"}}

        # 内置指数退避重试：每次失败等待 (attempt+1) 秒
        for attempt in range(self.max_retries):
            try:
                start = time.time()
                response = self.client.chat.completions.create(**kwargs)
                elapsed = time.time() - start
                self.call_count += 1
                self.total_prompt_tokens += response.usage.prompt_tokens
                self.total_completion_tokens += response.usage.completion_tokens
                self.logger.info(
                    f"[{self.call_count}] {self.model} | "
                    f"耗时:{elapsed:.1f}s | "
                    f"输入:{response.usage.prompt_tokens} | "
                    f"输出:{response.usage.completion_tokens}"
                )
                return response
            except Exception as e:
                self.logger.warning(f"第{attempt+1}次调用失败: {e}")
                if attempt < self.max_retries - 1:
                    time.sleep(1 * (attempt + 1))
                else:
                    raise

    def chat_text(self, messages: list[dict], **kwargs) -> str:
        """直接返回文本内容，适合对话和问答"""
        response = self.chat(messages, **kwargs)
        return response.choices[0].message.content

    def chat_json(self, messages: list[dict], **kwargs) -> dict:
        """强制 JSON 输出模式，返回解析后的 dict"""
        kwargs["response_format"] = {"type": "json_object"}
        response = self.chat(messages, **kwargs)
        content = response.choices[0].message.content
        return json.loads(content) if content else {}

    def chat_with_tools(
        self,
        messages: list[dict],
        tools: list[dict],
        tool_executor: Callable,
        max_rounds: int = 5,
        **kwargs
    ) -> str:
        """工具调用循环：模型 → 执行 → 结果 → 继续，最多 max_rounds 轮"""
        current_messages = messages.copy()
        for round_num in range(max_rounds):
            response = self.chat(messages=current_messages, tools=tools, **kwargs)
            message = response.choices[0].message
            if not message.tool_calls:
                return message.content
            current_messages.append(message)
            for tc in message.tool_calls:
                func_name = tc.function.name
                func_args = json.loads(tc.function.arguments)
                self.logger.info(f"调用工具: {func_name}({func_args})")
                try:
                    result = tool_executor(func_name, func_args)
                except Exception as e:
                    result = json.dumps({"error": str(e)})
                current_messages.append({
                    "role": "tool",
                    "tool_call_id": tc.id,
                    "content": result if isinstance(result, str) else json.dumps(result, ensure_ascii=False)
                })
        return "达到最大轮数"

    def get_usage_stats(self) -> dict:
        """按 DeepSeek 公开价格估算费用：输入 ¥0.14/M tokens，输出 ¥0.28/M tokens"""
        return {
            "model": self.model,
            "total_calls": self.call_count,
            "total_prompt_tokens": self.total_prompt_tokens,
            "total_completion_tokens": self.total_completion_tokens,
            "total_tokens": self.total_prompt_tokens + self.total_completion_tokens,
            "estimated_cost_cny": (self.total_prompt_tokens * 0.14 + self.total_completion_tokens * 0.28) / 1_000_000,
        }
```

import os, yaml
from pathlib import Path

_DIR = Path(__file__).parent

class PromptManager:
    """YAML 驱动的 Prompt 模板管理器：注册/渲染/持久化"""
    def __init__(self, config_path: str = None):
        # config_path 默认指向同目录下的 prompts.yaml
        if config_path is None:
            config_path = str(_DIR / "prompts.yaml")
        self.config_path = config_path
        self.templates = {}
        if os.path.exists(config_path):
            with open(config_path, "r", encoding="utf-8") as f:
                self.templates = yaml.safe_load(f) or {}

    def register(self, name: str, system: str, user_template: str):
        """注册新模板，存入内存（尚未写入磁盘）"""
        self.templates[name] = {
            "system": system,
            "user": user_template
        }

    def render(self, name: str, **kwargs) -> list[dict]:
        """按模板名渲染，返回可直接传入 API 的 messages 列表"""
        tpl = self.templates.get(name)
        if not tpl:
            raise ValueError(f"模板 '{name}' 不存在")
        return [
            {"role": "system", "content": tpl["system"]},
            {"role": "user", "content": tpl["user"].format(**kwargs)}
        ]

    def list_templates(self) -> list[str]:
        return list(self.templates.keys())

    def delete(self, name: str):
        self.templates.pop(name, None)

    def save(self, config_path: str = None):
        """将内存中的模板持久化回 YAML 文件"""
        if config_path is None:
            config_path = self.config_path
        with open(config_path, "w", encoding="utf-8") as f:
            yaml.dump(self.templates, f, allow_unicode=True)

#### evaluator.py

```python
from typing import Callable
from llm_client import LLMClient

class Evaluator:
    """Prompt 评估器：跑测试集 + 计算准确率 + A/B 对比"""
    def __init__(self):
        self.client = LLMClient()

    def run_test(
        self,
        test_cases: list[dict],
        system_prompt: str,
        judge_function: Callable[[str, str], bool],
        reasoning_effort: str | None = None,
        verbose: bool = False,
    ) -> dict:
        """
        在测试集上评估一个 Prompt 的准确率。
        test_cases: [{"input":..., "expected":...}]
        judge_function: 自定义判对函数，入参(实际输出, 期望值) → bool
        """
        results = []
        correct = 0
        # 逐条跑测试，记录结果
        for case in test_cases:
            messages = [
                {"role": "system", "content": system_prompt},
                {"role": "user", "content": case["input"]}
            ]
            output = self.client.chat_text(messages, temperature=0, reasoning_effort=reasoning_effort)
            is_correct = judge_function(output, case["expected"])
            if is_correct:
                correct += 1
            results.append({
                "input": case["input"],
                "output": output,
                "expected": case["expected"],
                "correct": is_correct
            })
            if verbose:
                mark = "[OK]" if is_correct else "[X]"
                print(f"  {mark} 输入: {case['input'][:30]} -> 输出: {output[:20]} | 期望: {case['expected']}")
        accuracy = correct / len(test_cases) if test_cases else 0
        if verbose:
            print(f"准确率: {accuracy:.0%} ({correct}/{len(test_cases)})")
        return {
            "accuracy": accuracy,
            "correct": correct,
            "total": len(test_cases),
            "results": results
        }

    def compare(
        self,
        test_cases: list[dict],
        prompt_v1: str,
        prompt_v2: str,
        judge_function: Callable
    ) -> dict:
        """A/B 对比：同一测试集分别跑两个 Prompt，输出准确率差异"""
        result_v1 = self.run_test(test_cases, prompt_v1, judge_function)
        result_v2 = self.run_test(test_cases, prompt_v2, judge_function)
        return {
            "v1": result_v1,
            "v2": result_v2,
            "improvement": result_v2["accuracy"] - result_v1["accuracy"]
        }
```

#### prompt_playground.py（入口）

```python
import sys, os, json
from pathlib import Path
from llm_client import LLMClient
from prompt_manager import PromptManager
from evaluator import Evaluator

_PROJECT = Path(__file__).parent

# 全局实例：脚本启动时一次性初始化
client = LLMClient()
pm = PromptManager()
evaluator = Evaluator()
TEST_CASES_PATH = _PROJECT / "test_cases.json"

def load_test_cases(path=None):
    """加载测试用例：优先用传入路径，fallback 到默认 test_cases.json"""
    if path and os.path.exists(path):
        with open(path, "r", encoding="utf-8") as f:
            return json.load(f)
    if os.path.exists(TEST_CASES_PATH):
        with open(TEST_CASES_PATH, "r", encoding="utf-8") as f:
            return json.load(f)
    return []

def interactive_mode():
    """CLI 主循环：注册 /templates /eval /benchmark 等交互命令"""
    print("Prompt Playground v2.0")
    print(f"默认测试用例: {TEST_CASES_PATH} ({len(load_test_cases())}条)")
    print("命令: /templates /new /eval /usage /save /help /benchmark /quit")

    while True:
        cmd = input("\n> ").strip()
        if cmd == "/quit":
            break
        elif cmd == "/templates":
            # 列出所有已注册的模板名称
            for name in pm.list_templates():
                print(f"  - {name}")
        elif cmd == "/new":
            # 创建新模板：输入模板名 + System Prompt + User 模板
            name = input("模板名: ")
            system = input("System Prompt: ")
            user_tpl = input("User 模板 (用 {var} 做变量): ")
            pm.register(name, system, user_tpl)
            pm.save()
            print(f"模板 '{name}' 已保存")
        elif cmd == "/eval":
            # 单模板评估：选择模板 → 加载测试集 → 逐条判对 → 输出准确率
            template_name = input("测试哪个模板?: ")
            if template_name not in pm.list_templates():
                print("模板不存在")
                continue
            test_file = input("测试用例 JSON 路径 (留空默认): ")
            test_cases = load_test_cases(test_file.strip() if test_file.strip() else None)
            if not test_cases:
                print("无测试用例，跳过")
                continue
            system_prompt = pm.templates[template_name]["system"]
            evaluator.run_test(
                test_cases,
                system_prompt,
                judge_function=lambda out, exp: exp in out,
                verbose=True
            )
        elif cmd == "/benchmark":
            # 全模板横向对比：所有模板跑同一测试集，看哪个准确率最高
            names = pm.list_templates()
            if not names:
                print("无模板")
                continue
            test_file = input("测试用例 JSON 路径 (留空默认): ")
            test_cases = load_test_cases(test_file.strip() if test_file.strip() else None)
            if not test_cases:
                print("无测试用例，跳过")
                continue
            for name in names:
                sp = pm.templates[name]["system"]
                result = evaluator.run_test(test_cases, sp, lambda out, exp: exp in out)
                print(f"  [{name}] 准确率: {result['accuracy']:.0%} ({result['correct']}/{result['total']})")
        elif cmd == "/usage":
            # 查看累计 Token 消耗和估算费用
            stats = client.get_usage_stats()
            print(json.dumps(stats, indent=2, ensure_ascii=False))
        elif cmd == "/help":
            print("命令列表:")
            print("  /templates  - 列出所有模板")
            print("  /new        - 创建新模板")
            print("  /eval       - 测试单个模板效果")
            print("  /benchmark  - 对比所有模板效果")
            print("  /usage      - 查看用量统计")
            print("  /save       - 保存配置")
            print("  /quit       - 退出")
        elif cmd.startswith("/"):
            print(f"未知命令: {cmd}")
        else:
            # 直接输入文本走这里：默认使用第一个模板渲染 messages
            if pm.list_templates():
                name = pm.list_templates()[0]
                tpl = pm.templates.get(name, {})
                user_tpl = tpl.get("user", "{input}")
                if "{input}" in user_tpl or "{text}" in user_tpl:
                    messages = pm.render(name, input=cmd, text=cmd, context=cmd, question=cmd)
                else:
                    messages = [{"role": "user", "content": cmd}]
            else:
                messages = [{"role": "user", "content": cmd}]
            reply = client.chat_text(messages)
            print(f"\n{reply}")

if __name__ == "__main__":
    interactive_mode()
```

| 命令 | 演示内容 | 预期效果 |
|------|----------|----------|
| `/templates` | 列出所有预置模板 | 显示 6 个模板（情感分类-v1/v2/v3、实体提取-v1/v2、语言识别） |
| `/eval` | 测试单个模板 | 先输入模板名（如 `情感分类-v1`），回车使用默认测试集 → 输出准确率 + 逐条结果（正确/错误） |

| `/usage` | 查看用量 | 显示总调用次数、Token 消耗、估算费用 |
| 直接输入文本 | 自由聊天 | 用第一个可用模板对话，适合快速验证模板效果 |
